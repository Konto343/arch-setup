# My Arch Setup:
My arch setup for new computers. Includes my personal dotfile and scripts as well.
I nicked name this setup: "Kool"

This setup is supposed to be:
- Fast
- Minimalist/Clean
- Resource Effecient
- "Just Works"
- Privacy-Focused

## In this setup:
- Terminal: 		Kitty
- Shell:		Fish
- Notifier:		Mako
- App Launcher:		Wofi
- File Explorer:	Thunar
- Window Compsitor: 	Hyprland
- Firewall:		Ufw
 	**+ Waybar,Hyprlock,Hypridle,Waypaper,Hyprpicker**
- Browser: 		LibreWolf
- Communications: 	Vesktop, Hexchat
- Mail: 		BetterBird
- Passwords: 		KeePassXc
- Recording:		Obs
- Code Editor:		VsCodium
- Text Editor:		Neovim, Vim
- Video Player:		Mpv
- Notes:		Obsidian
- Video Editor:		OpenShot
- Office Suite:		LibreOffice (Still)
- Image Editor:		Gimp,Inkscape

## Usage:
Copy the Repo, and run the "install_setup.sh" script, and copy over the "home" folder to your
home directory.
